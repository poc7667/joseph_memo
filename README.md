# 分享你的照片、音樂、與留言

請將分享的內容放在一個資料夾

資料夾名稱將會當做您的`顯示姓名`

將要分享的照片放在資料夾中。(optional)

接著建立一個文字檔txt，把訴說的話語記錄於其中。

# words.txt


`第一行文字`將作為標題使用。

如果你有要提供 Youtube 影片音樂，請直接複製網址貼在裡面。
